## 🧙‍♂️ Harry Potter ToDo List

Welcome to the magical world of **Harry Potter**! This is a **ToDo List** app with a wizarding theme, where you can manage your tasks with some ✨magic✨.

---

## 🌟 Features

- Add tasks with a **magical vibe**
- Mark tasks as **Done** with a spell effect
- Delete tasks when completed 🧹
- Background **Harry Potter-inspired music** 🎵
- Magical animations and styling 🪄
- Wizard-themed design and fonts
